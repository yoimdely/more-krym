# ЖК «Море» (Крым) — комплект файлов

Распакуйте содержимое архива в корень вашего текущего проекта Vite (куда вы уже заливали «Просторы Крыма»).
Будут созданы/перезаписаны пути:

- `src/App.jsx` — новая страница под ЖК «Море`.
- `public/privacy.html`, `public/consent.html` — юридические страницы.
- `public/sitemap.xml`, `public/robots.txt` — для SEO и вебмастеров.
- `public/media/*.jpg` — 8 заглушек изображений; замените их на реальные фото ЖК «Море`.

После замены — обычный `npm run build` и публикация на GitHub Pages.
